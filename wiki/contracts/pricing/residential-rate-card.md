# Pricing: Residential Rate Card

---

## ⚡ ACTION STEPS (When Generating Quote)

**AI Returns:**
1. **Room Analysis** — Size → HP recommendation (with KL climate factor)
2. **3 Options** — Budget / Best Value / Premium (with prices, lead times, availability)
3. **Installation Availability** — Next available slots + assigned technician
4. **Draft Message** — Professional quote with options
5. **Action Buttons** — [SEND QUOTE] [EDIT] [REQUEST SURVEY] [CALL]

**Owner:** Reply with SEND QUOTE / EDIT / REQUEST SURVEY / CALL

**If SEND QUOTE:** Quote sent to customer, follow-up scheduled in 24hrs
**If EDIT:** Owner modifies → sent
**If REQUEST SURVEY:** Site survey offered instead
**If CALL:** Reminder to call for complex quotes

## Payment Terms
- **Service:** Cash, DuitNow, TNG eWallet
- **Invoicing:** Same day (WhatsApp invoice)
- **Credit Terms:** Not available for residential (prepaid only)

## Discounts
- **Annual Contract (2x service/year):** 10% off per service
- **3+ Units (same visit):** 5% off total
- **Referral:** RM 20 credit for each new customer referral
